<p><strong>Module 1: Optimizing Windows Desktop Management in the Enterprise</strong> - This module explains key desktop manageability challenges and  identifies MDOP applications that can address those challenges.</p>
<p><strong>Module 2: Controlling Group  Policy Objects Using Advanced Group Policy Management</strong> - This module explains how to use AGPM to manage controlled group policy  objects.</p>
<p><strong>Module 3: Monitoring Desktops  Using Microsoft Desktop Error Monitoring</strong> - This module explains how Desktop Error Monitoring can be configured to  provide a centralized collection location of computer events and error messages.</p>
<p><strong>Module 4: Managing Software  Inventory Using the Asset Inventory Service</strong> - This module explains how to use the Microsoft Asset Inventory Service  to manage software inventory.</p>
<p><strong>Module 5: Repairing and  Diagnosing Computer Systems Using the Diagnostics and Recovery Toolset</strong> - This module explains how to use the Diagnostics and Recover Toolset to  recover computer systems.</p>
<p><strong>Module 6: Introduction to the  Microsoft Application Virtualization Management System</strong> - This module explains application virtualization concepts and how the  components of Microsoft Application Virtualization Management system address  the various phases of the Application Management Lifecycle.</p>
<p><strong>Module 7: Application  Virtualization Management Architecture</strong> - This module explains the general architecture of Application Virtualization  Management.</p>
<p><strong>Module 8: Planning and Deploying  the Application Virtualization Management System</strong> - This module explains how to plan the supporting infrastructure and  determine the appropriate deployment scenario and installation for an  Application Virtualization Management System.</p>
<p><strong>Module 9: Planning and  Installing the Application Virtualization Client</strong> - This module explains how to plan for the deployment of the Application  Virtualization Client and then install and perform client configuration tasks.</p>
<p><strong>Module 10: Administering the  Application Virtualization Management Server</strong> - This module explains how to use the Application Virtualization  Management Console to publish applications into the application virtualization  environment and configure active upgrades.</p>
<p><strong>Module 11: Advanced Application  Virtualization Administration Tasks</strong> - This module explains how to create new provider policies to apply specific settings such as license enforcement to specific users.</p>
<p><strong>Module 12: Planning and  Deploying the Application Virtualization Sequencer</strong> - This module explains how to perform various sequence scenarios using  the Application Virtualization Sequencer.</p>
<p><strong>Module 13: Advanced Sequencing</strong> - This  module explains how to perform advanced sequence scenarios.</p>
