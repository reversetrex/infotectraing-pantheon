<p><strong>Module 1: SharePoint as a Developer Platform</strong></p>
<blockquote>This module examines different approaches that can be used to develop applications with SharePoint Server 2013 the scenarios in which each approach might be appropriate.</blockquote>
<p>Lessons</p>
<ul>
  <li>Introducing the SharePoint Developer Landscape</li>
  <li>Choosing Approaches to SharePoint Development</li>
  <li>Understanding SharePoint 2013 Deployment and Execution Models</li>
</ul>
<p><strong>Module 2: Working with SharePoint Objects</strong></p>
<blockquote>This module introduces the server-side SharePoint object model and how the core classes relate to sites and collections. The server-side SharePoint object model provides a core set of classes that represent different items in the logical architecture of a SharePoint deployment. Students also learn how manage permissions for server-side code.</blockquote>
<p>Lessons</p>
<ul>
  <li>Understanding the SharePoint Object Hierarchy</li>
  <li>Working with Sites and Webs</li>
  <li>Working with Execution Contexts</li>
</ul>
<p><strong>Module 3: Working with Lists and Libraries</strong></p>
<blockquote>This module explains how to interact with lists and libraries. Students learn about how to work with lists and libraries programmatically using the server-side SharePoint object model and how to use query classes and LINQ to SharePoint to query and retrieve data from SharePoint lists. Student also learn how to efficiently work with lists that contain large numbers of items.</blockquote>
<p>Lessons</p>
<ul>
  <li>Using List and Library Objects</li>
  <li>Querying and Retrieving List Data</li>
  <li>Working with Large Lists</li>
</ul>
<p><strong>Module 4: Designing and Managing Features and Solutions</strong></p>
<blockquote>This module examines creating and deploying custom Developing a SharePoint solutions and features. The students also learn how and when to use sandbox solutions.</blockquote>
<p>Lessons</p>
<ul>
  <li>Understanding Features and Solutions</li>
  <li>Configuring Features and Solutions</li>
  <li>Working with Sandboxed Solutions</li>
</ul>
<p><strong>Module 5: Working with Server-Side Code</strong></p>
<blockquote>This module describes how to develop and deploy Web Parts and event receivers in a solution.</blockquote>
<p>Lessons</p>
<ul>
  <li>Developing Web Parts</li>
  <li>Using Event Receivers</li>
  <li>Using Timer Jobs</li>
  <li>Storing Configuration Data</li>
</ul>
<p><strong>Module 6: Managing Identity and Permissions</strong></p>
<blockquote>This module describes how manage permissions through code and customize authentication using custom claims providers.</blockquote>
<p>Lessons</p>
<ul>
  <li>Understanding Identity Management in SharePoint 2013</li>

<p>Lessons</p>
<ul>

</ul>
