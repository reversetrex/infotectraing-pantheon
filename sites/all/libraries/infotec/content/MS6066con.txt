<p><strong>Module 1: Introduction to .NET  Application Development for Windows Mobile 6</strong> - This module provides an overview of  the Windows Mobile 6 platform and mobile application development by using  Visual Studio 2005 and the .NET Compact Framework 2.0.</p>
<p><strong>Module 2: Developing Adaptable  Applications for Different Device Capabilities</strong> - This module explains how to develop  adaptable applications for mobile devices with different screen designs and  hardware capabilities.</p>
<p><strong>Module 3: Creating a Local Data  Store by Using SQL Server 2005 Compact Edition</strong> - This module explains how to create a  SQL Server 2005 Compact Edition database and store and retrieve data in the  database.</p>
<p><strong>Module 4: Synchronizing SQL Server 2005 Compact Edition with SQL Server 2005</strong> - This module explains how to configure the server-side components for replication and then program the device for replication</p>
<p><strong>Module 5: Developing Loosely  Connected Windows Mobile Applications</strong> - This module describes how to detect  network connection availability and queue Web service calls until a network is  available.</p>
<p><strong>Module 6: Planning and Implementing  Device and Application Security</strong> - This module describes how to  identify security vulnerabilities and implement application security features  to protect data.</p>
<p><strong>Module 7: Enhancing Windows Mobile 6-based Applications with Multimedia and Location-aware Capabilities</strong> - This module explains how to accept  ink input, play sounds, and write programs that read data from a GPS receiver.</p>
<p><strong>Module 8: Integrating Applications  With Office Mobile</strong> - This module describes how to program  Office Mobile features, implement messages and emails, and make a telephone call.</p>
<p><strong>Module 9: Packaging and Deploying  Applications</strong> - This  module explains how to package and deploy Windows Mobile applications.</p>
