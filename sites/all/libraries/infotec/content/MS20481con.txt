<p><strong>Module 1: Overview of the Windows 8.1 Platform and Windows Store Apps</strong></p>
<blockquote>This module introduces you to the Windows 8.1 user experience and features, Windows Store apps, and the new user interface. Understanding how Windows Store apps look and operate is the key to understanding the principles that you will use to develop your own Windows Store apps. This will help you to develop apps that are more intuitive, compelling, and useful.</blockquote>
<p>Lessons</p>
<ul>
  <li>Introduction to the Windows 8.1 Platform</li>
  <li>Windows 8 UI Principles</li>
  <li>WinRT and Language Projections</li>
</ul>
<p><strong>Module 2: Single-Page Apps and the MVVM Design Pattern</strong></p>

<p>Lessons</p>
<ul>
  <li>Single-Page Applications</li>
  <li>The MVVM Design Pattern</li>
</ul>
<p><strong>Module 3: Using WinJS</strong></p>
<blockquote>In this module, you will explore the WinJS library and some commonly used WinJS APIs. In particular, you will learn about the WinJS.Promise object and how to use it, the WinJS.Application namespace and the functionality it contains, and the WinJS.Utilities functions. Becoming familiar with all those APIs will help you to create reusable and robust apps for the Windows platform.</blockquote>
<p>Lessons</p>
<ul>
  <li>The WinJS Library</li>
