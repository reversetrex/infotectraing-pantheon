<p><strong>Module 4: Manipulating Data Sets</strong> - This module explores data sets to a greater depth, including the use of alternative data sources and interacting with a data set through the use of parameters. Students learn how to dynamically modify the data set underlying a data region by allowing parameters to be sent to the underlying query. They also learn to use best practices to implement static and dynamic parameter lists when interacting with queries and stored procedures.</p>

<p><strong>Lessons</strong></p>
<ul>
  <li>Defining Report Data </li>
  <li>Using Parameters and Filters </li>
  <li>Using Parameter Lists </li>
</ul>

</ul>
