<p><strong>Module 1: Introduction to Designing a Microsoft Exchange Server 2010 Deployment</strong></p>

<p>This module explains how to gather  business and technical requirements, and information about the current  messaging system, to prepare for designing a messaging system. It also provides  an overview of Service Level Management and how it can impact the design of the  messaging system.</p>

<ul>
  <li>Gathering  Business Requirements</li>
  <li>Identifying  Additional Requirements</li>
  <li>Introduction  to Service Level Management</li>
  <li>Analyzing  the Current Messaging Environment</li>
</ul>

<p><strong>Module 2:  Designing Microsoft Exchange Server 2010 Integration with the Current Infrastructure</strong></p>

<p>This module explains how to design  the integration of Exchange Server with the current infrastructure. It  describes how to design the network infrastructure, the Active Directory  infrastructure, and the DNS infrastructure. Additionally, this module explains  how to plan the administration of Exchange Server 2010.</p>

<ul>
  <li>Designing  the Network Infrastructure</li>
  <li>Designing  the Active Directory Infrastructure</li>
  <li>Designing  the DNS Infrastructure</li>
  <li>Planning  Exchange Server Administration</li>
</ul>

<p><strong>Module 3:  Planning and Deploying Mailbox Services</strong></p>

<p>This module explains how to design  the deployment of the Mailbox server services in Exchange Server 2010. It  provides an overview of the Mailbox services in Exchange Server 2010. The  module then describes how to design Mailbox server, recipient management, and a  public folder architecture.</p>

<ul>
  <li>Overview  of Mailbox Services in Exchange Server 2010</li>
  <li>Designing  Mailbox Servers</li>
  <li>Designing  Recipient Management</li>
  <li>Designing  Public Folder Architecture</li>
  <li>Lab: Planning and Deploying Mailbox Services</li>
</ul>

<ul>

  <li>Designing  the Client Access Server Deployment</li>

<p><strong>Module 5: Planning and Deploying Message  Transport in Exchange Server 2010</strong></p>

<ul>

  <li>Designing  Hub Transport Servers</li>
  <li>Designing  the Message Routing Perimeter</li>
</ul>

<p><strong>Module 6:  Planning and Deploying Messaging Security</strong></p>

<p>This module explains how to plan and  deploy messaging security. It describes how to design message security  restrictions, SMTP connector security, and explains how to secure message  routing between partner organizations. The module then describes how to design  antivirus and anti-spam solutions.</p>

<ul>
  <li>Designing  Message Security</li>
  <li>Designing  Antivirus and Anti-Spam Solutions</li>
</ul>

<p><strong>Module 7: Planning and Deploying Messaging Compliance</strong></p>

<p>This module explains how to plan and  deploy a messaging policy and compliance solution. It describes how to design  transport rules, message classifications, and message moderation. Then the  module describes how to design the AD RMS integration with Exchange Server  2010. Finally, the module explains how to design message journaling, archiving,  and messaging records management.</p>

<ul>
  <li>Designing  Transport Compliance</li>
  <li>Designing  AD RMS Integration with Exchange Server 2010</li>
  <li>Designing  Message Journaling and Archiving</li>
  <li>Designing  Messaging Records Management</li>
</ul>

<p><strong>Module 8: Planning and Deploying High Availability</strong></p>

<ul>

</ul>
