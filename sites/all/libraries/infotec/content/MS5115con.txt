<p><strong> Unit 1: Installing Windows Vista</strong></p>
<p> This Unit describes the new features of the Windows Vista operating system and then explains how to perform a clean installation. This Unit also explains how to install and configure Windows Vista device drivers.</p>
<p> Units </p>
<ul>
  <li> Introduction to Windows Vista. </li>
  <li>Performing a Clean Installation of Windows Vista. </li>
  <li>Installing and Configuring Windows Vista Device Drivers. </li>
</ul>
<p> Lab: Installing Windows Vista </p>
<ul>
  <li>Verifying Hardware Requirements </li>
  <li>Installing Windows Vista from DVD </li>
  <li>Managing Device Drivers </li>
</ul>
<p> After completing this Unit, students will be able to: </p>
<ul>
  <li> Describe the Windows Vista operating system. </li>
  <li>Perform a clean installation of Windows Vista. </li>
  <li>Install and configure Windows Vista device drivers. </li>
</ul>
