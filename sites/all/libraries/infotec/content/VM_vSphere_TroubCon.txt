<ul>
  <li> Install  and use the vMA appliance</li>

  <li> Configure  ESXi technical support mode and SSH access</li>
</ul>

</ul>

</ul>

<p><strong>4. Management Troubleshooting</strong></p>

</ul>

</ul>

<p><strong>8. DRS Cluster Troubleshooting</strong></p>
