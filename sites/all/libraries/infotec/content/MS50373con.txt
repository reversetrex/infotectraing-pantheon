<p><strong>Module 1:  Installing Essentials 2010</strong></p>
<blockquote>
  <p>  This  module describes the architecture of System Center Essentials 2010. It goes on  to discuss hardware and software requirements, as well as the Essentials 2010  installation process.</p>
</blockquote>
<p><strong>Module 2:  Configuring Essentials</strong></p>
<blockquote>
  <p>  This  module explains how to perform the initial configuration tasks after installing  System Center Essentials 2010.</p>
</blockquote>
<p><strong>Module 3:  Managing Computers</strong></p>
<blockquote>
  <p>  This  module explains how to prepare, add and group computers for management with  Essentials 2010. It goes on to demonstrate how to retrieve settings and  inventory of managed computers.</p>
</blockquote>
<p><strong>Module 4:  Managing Updates</strong></p>
<blockquote>
  <p>  The  module explains how to approve and deploy updates with System Center Essentials  2010. It includes information on how to automate the update management process  if desired.</p>
</blockquote>
<p><strong>Module 5:  Deploying Software</strong></p>
<blockquote>
  <p>  This  module explains how to install, update and uninstall software to managed  computers with System Center Essentials 2010.</p>
</blockquote>
<p><strong>Module 6:  Monitoring Concepts</strong></p>
<blockquote>
  <p>  This  module explains how to configure, administer and optimize the monitoring  functionality in Essentials to monitor Windows computers and network devices.</p>
</blockquote>
<p><strong>Module 7:  Reporting</strong></p>
<blockquote>
  <p>  The  module describes the architecture and functionality of the reporting feature in  System Center Essentials 2010. It goes on to describe the data retention and  grooming settings in the reporting feature.</p>
</blockquote>
<p><strong>Module 8:  Virtualization Management</strong></p>
<blockquote>
  <p>  The  module describes the architecture and functionality of the virtualization  management feature in System Center Essentials 2010.</p>
</blockquote>
<p><strong>Module 9:  Virtual Machine Templates and Provisioning</strong></p>
<blockquote>
  <p>  This  module describes how to create virtual machine templates to streamline the  provisioning process. It goes on to explain the process for converting an  existing VMware virtual machine to the Hyper-V platform.</p>
</blockquote>
<p><strong>Module 10:  Advanced Monitoring and Management Pack Authoring</strong></p>
<blockquote>
  <p>  This  module describes the advanced monitoring functionality available in System  Center Essentials 2010. You will also learn important management pack authoring  concepts for creating custom management packs for your line-of-business applications.</p>
</blockquote>
<p><strong>Module 11:  Advanced Troubleshooting and Log Analysis</strong></p>
<blockquote>
  <p>  This  module covers the log files, tools and techniques to troubleshoot System  Centers Essentials 2010 issues in depth. The troubleshooting processes for  common scenarios related to Essentials features are described in detail.</p>
</blockquote>
<p><strong>Module 12:  Essentials Internals</strong></p>
<blockquote>
  <p>This module presents  information on Essentials core functions and integration with its foundational  technologies (WSUS, SCVMM 2008 R2 and OpsMgr R2) in greater detail. It examines  internal functions such as group synchronization behavior between Essentials  and WSUS, WSUS certificates and how they are leveraged in Essentials,  deployment and inventory functions.</p>
</blockquote>
