<p><strong>Module 1: Introduction to Mastering Microsoft Project - This module provides an overview of how the features of Project relate to the job of the project manager.</strong></p>
<ul>
  <li>Describe how Project relates to the discipline of Project management.</li>
  <li>Know the significant new features of   Project 2010.</li>
  <li>Navigate to the primary views available using the Ribbon</li>
  <li>Choose Views that display task, resource, or assignment information.</li>
  <li>Select table within views to change the information that is available to see and edit.</li>
  <li>Relate the features of Project to the 5 steps for building a plan in Project.</li>
</ul>
<p><strong>Module 2: A Quick and Easy Overview of Managing with Project - This module demonstrates the steps required to create and use Project through the life cycle of a project.</strong></p>
<ul>

  <li>Enter project tasks.</li>
  <li>Sequence the tasks.</li>
  <li>Define resources.</li>
  <li>Estimate Task duration and assign resources.</li>
  <li>Baseline the project.</li>
  <li>Track project progress.</li>
</ul>
<p><strong>Module 3: Setting Up a Project - This module explains how to create a new project and establish the basic constraints that Project will use for its calculations.</strong></p>
<ul>
  <li>Use multiple methods to create a new project, including opening an Excel file and a SharePoint Tasks list.</li>
  <li>Establish one or more calendars to constrain resource availability.</li>
  <li>Configure Project to calculate the schedule from the Start Date forward, or from the Finish Date backward.</li>
</ul>
<p><strong>Module 4: Manually Schedule vs. Auto Schedule - This module explains how to manually schedule project tasks and how to leverage the auto schedule feature.</strong></p>
<ul>
  <li>Describe which project functions are turned off for tasks using Manually Schedule mode.</li>
  <li>Change the task mode from Manually Schedule to Auto Schedule and back.</li>
  <li>Identify tasks that are in Manually Schedule mode by the task mode column and shape on the Gantt chart.</li>
  <li>Describe situations that are particularly appropriate for using Manually Schedule.</li>
  <li>Describe the limitations that a user must be aware of when using Manually Schedule mode.</li>
</ul>

<p><strong>Module 6: Identifying Task Relationships - This module explains the rules for establishing dependency links between tasks and presents the features to use Project to establish and display these dependencies.</strong></p>

<p><strong>Module 7: Defining resources within Project - This module explains how to enter resources and specific resource information in Microsoft Project and assign resources to specific tasks.</strong></p>

  <li>Record the limit of availability for each type of resource by establishing a resource calendar and defining the   maximum units of that resource.</li>
</ul>
