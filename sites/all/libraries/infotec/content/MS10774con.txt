<p><strong>Module 1: Introduction to Microsoft SQL Server 2012</strong></p>
<p>This module introduces the SQL Server platform and major tools. It discusses editions, versions, tools used to query, documentation sources, and the logical structure of databases.</p>
<ul>
  <li>Introducing Microsoft SQL Server 2012</li>
  <li>Getting Started with SQL Server Management Studio</li>
</ul>
<p><strong>Module 2: Getting Started with SQL Azure</strong></p>
<p>This module introduces you to the concepts of SQL Azure. If the virtual machines in your classroom are able to connect to the internet and you have a Windows Azure account you may be able to connect to your Azure server and database. Many of the labs in the rest of this course are enabled for you to perform the lab while connected to your own Azure database in the cloud.</p>
<ul>
  <li>Overview of SQL Azure</li>
  <li>Working with SQL Azure</li>
</ul>
<p><strong>Module 3: Introduction to T-SQL Querying</strong></p>
<p>This module introduces Transact SQL as the primary querying language of SQL Server. It discusses the basic structure of T-SQL queries, the logical flow of a SELECT statement, and introduces concepts such as predicates and set-based operations.</p>
<ul>
  <li>Introducing T-SQL</li>
  <li>Understanding Sets</li>
  <li>Understanding Predicate Logic</li>
  <li>Understanding the Logical Order of Operations in SELECT statements</li>
</ul>
<p><strong>Module 4: Writing SELECT Queries</strong></p>
<p>This module introduces the fundamentals of the SELECT statement, focusing on queries against a single table.</p>
<ul>
  <li>Writing Simple SELECT Statements</li>
  <li>Eliminating Duplicates with DISTINCT</li>
  <li>Using Column and Table Aliases</li>
  <li>Writing Simple CASE Expressions</li>
</ul>
<p><strong>Module 5: Querying Multiple Tables</strong></p>
<p>This module explains how to write queries which combine data from multiple sources in SQL Server. The module introduces the use of JOINs in T-SQL queries as a mechanism for retrieving data from multiple tables.</p>
<ul>
  <li>Understanding Joins</li>
  <li>Querying with Inner Joins</li>
  <li>Querying with Outer Joins</li>
  <li>Querying with Cross Joins and Self Joins</li>
</ul>
<p><strong>Module 6: Sorting and Filtering Data</strong></p>
<p>This module explains how to enhance queries to limit the rows they return, and to control the order in which the rows are displayed. The module also discusses how to resolve missing and unknown results.</p>
<ul>
  <li>Sorting Data</li>
  <li>Filtering Data with a WHERE Clause</li>
  <li>Filtering with the TOP and OFFSET-FETCH Options</li>
  <li>Working with Unknown and Missing Values</li>
</ul>
<p><strong>Module 7: Working with SQL Server 2012 Data Types</strong></p>
<p>This module explains the data types SQL Server uses to store data. It introduces the many types of numeric and special-use data types. It also explains conversions between data types, and the importance of type precedence.</p>
<ul>
  <li>Introducing SQL Server 2012 Data Types</li>
  <li>Working with Character Data</li>
  <li>Working with Date and Time Data</li>
</ul>
<p><strong>Module 8: Using Built-In Functions</strong></p>
<p>This module introduces the use of functions that are built in to SQL Server Denali, and will discuss some common usages including data type conversion, testing for logical results and nullability.</p>
<ul>
  <li>Writing Queries with Built-In Functions</li>
  <li>Using Conversion Functions</li>
  <li>Using Logical Functions</li>
  <li>Using Functions to Work with NULL</li>
</ul>
<p><strong>Module 9: Grouping and Aggregating Data</strong></p>
<p>This module introduces methods for grouping data within a query, aggregating the grouped data and filtering groups with HAVING. The module is designed to help the student grasp why a SELECT   clause has restrictions placed upon column naming in the GROUP BY clause as well as which columns may be listed in the SELECT clause.</p>
<ul>
  <li>Using Aggregate Functions</li>
  <li>Using the GROUP BY Clause</li>
  <li>Filtering Groups with HAVING</li>
</ul>
<p><strong>Module 10: Using Subqueries</strong></p>
<p>This module will introduce the use of subqueries in various parts of a SELECT statement. It will include the use of scalar and multi-result subqueries, and the use of the IN and   EXISTS operators.</p>
<ul>
  <li>Writing Self-Contained Subqueries</li>
  <li>Writing Correlated Subqueries</li>
  <li>Using the EXISTS Predicate with Subqueries</li>
</ul>
<p><strong>Module 11: Using Table Expressions</strong></p>
<p>This module introduces T-SQL expressions which return a valid relational table, typically for further use in the query. The module discusses views, derived tables, common table expressions and inline table-valued functions.</p>
<ul>
  <li>Using Derived Tables</li>
  <li>Using Common Table Expressions</li>
  <li>Using Views</li>
  <li>Using Inline Table-Valued Functions</li>
</ul>
<p><strong>Module 12: Using Set Operators</strong></p>
<p>This module introduces operations involving multiple sets of data. It will cover the use of the UNION, UNION ALL, APPLY, CROSS APPLY, OUTER APPLY operators as well as the EXCEPT and INTERSECTS operators.</p>
<ul>
  <li>Writing Queries with the UNION Operator</li>
  <li>Using EXCEPT and INTERSECT</li>
  <li>Using APPLY</li>
</ul>
<p><strong>Module 13: Using Window Ranking, Offset and Aggregate Functions</strong></p>
<p>This module introduces window functions including ranking, aggregate and offset functions. Much of this functionality is new to SQL Server 2012. It will cover the use of T-SQL functions such as ROW_NUMBER, RANK, DENSE_RANK, NTILE, LAG, LEAD, FIRST_VALUE and LAST_VALUE to perform calculations against a set, or window, of rows.</p>
<ul>
  <li>Creating Windows with OVER</li>
  <li>Exploring Window Functions</li>
</ul>
<p><strong>Module 14: Pivoting and Grouping Sets</strong></p>
<p>This module discusses techniques for pivoting data in T-SQL as well to introduce the fundamentals of the GROUPING SETS clause. It will also cover the use of GROUP BY ROLLUP and GROUP BY CUBE syntax in SQL Server 2012.</p>
<ul>
  <li>Writing Queries with PIVOT and UNPIVOT</li>
  <li>Working with Grouping Sets</li>
</ul>
<p><strong>Module 15: Querying SQL Server Metadata</strong></p>
<p>This module introduces the use of SQL Server system objects in T-SQL queries. It will cover the use of system catalog views, system stored procedures, system functions, and dynamic management objects.</p>
<ul>
  <li>Querying System Catalog Views and Functions</li>
  <li>Executing System Stored Procedures</li>
  <li>Querying Dynamic Management Objects</li>
</ul>
<p><strong>Module 16: Executing Stored Procedures</strong></p>
<p>This module introduces the use of existing stored procedures in a T-SQL querying environment. It discusses the use of EXECUTE, how to pass input and output parameters to a procedure, and how to invoke system stored procedures.</p>
<ul>
  <li>Querying Data with Stored Procedures</li>
  <li>Passing Parameters to Stored Procedures</li>
  <li>Creating Simple Stored Procedures</li>
  <li>Working with Dynamic SQL</li>
</ul>
<p><strong>Module 17: Programming with T-SQL</strong></p>
<p>This module provides a basic introduction to T-SQL programming concepts and objects. It discusses batches, variables, control of flow elements such as loops and conditionals, how to create and execute dynamic SQL statements, and how to use synonyms.</p>
<ul>
  <li>T-SQL Programming Elements</li>
  <li>Controlling Program Flow</li>
</ul>
<p><strong>Module 18: Implementing Error Handling</strong></p>
<p>This module introduces the use of error handlers in T-SQL code. It will introduce the difference between compile errors and run-time errors, and will cover how errors affect batches. The module will also cover how to control error handling using TRY/CATCH blocks, the use of the ERROR class of functions, and the use of the new THROW statement.</p>
<ul>
  <li>Using TRY / CATCH Blocks</li>
  <li>Working with Error Information</li>
</ul>
<p><strong>Module 19: Implementing Transactions</strong></p>
<p>This module introduces the concepts of transaction management in SQL Server. It will provide a high-level overview of transaction properties, cover the basics of marking transactions with BEGIN, COMMIT and ROLLBACK.</p>
<ul>
  <li>Transactions and the Database Engine</li>
  <li>Controlling Transactions</li>
</ul>
<p><strong>Module 20: Improving Query Performance</strong></p>
<p>This module introduces the concepts of system resource usage and the performance impact of querying SQL Server 2012. It will cover, at a high level, the use of indexes in SQL Server, the use of execution plans in SQL Server Management Studio, and the use of SET options to view system resource usage when executing queries. It will also compare set-based operations with cursor-based operations.</p>
<ul>
  <li>Factors in Query Performance</li>
  <li>Displaying Query Performance Data</li>
</ul>
