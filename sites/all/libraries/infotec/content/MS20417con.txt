<p><strong>Module 1: Installing and Configuring Windows Server 2012</strong></p>
<p>This module explains how to install and configure Windows Server 2012. It specifically covers requirements and considerations for installation and the installation of roles and installation types such as Server Core. It also covers the implementation and configuration of remote management of servers running Windows Server 2012.</p>
<ul>
  <li>Installing Windows Server 2012 R2</li>
  <li>Configuring Windows Server 2012 R2 and Windows Server 2012</li>
  <li>Configuring Remote Management for Windows Server 2012 R2 and Windows Server 2012</li>
</ul>
<p><strong>Module 2: Managing Windows Server 2012 by Using Windows PowerShell</strong></p>
<p>This module explains how to use Windows PowerShell to manage Windows Server 2012. It will provide an overview of Windows PowerShell and the Windows PowerShell Integrated Scripting Environment (ISE). It will also cover Windows PowerShell in the context of AD DS and general server management.</p>
<ul>
  <li>Overview of Windows PowerShell</li>
  <li>Using Windows PowerShell to Manage AD DS</li>
  <li>Managing Servers by Using Windows PowerShell</li>
</ul>
<p><strong>Module 3: Managing Storage in Windows Server 2012</strong></p>
<p>This module explains how to configure storage in Windows Server 2012.</p>
<ul>
  <li>Storage Features in Windows Server 2012</li>
  <li>Configuring iSCSI Storage</li>
  <li>Configuring Storage Spaces in Windows Server 2012</li>
  <li>Configuring BranchCache in Windows Server 2012</li>
</ul>
<p><strong>Module 4: Implementing Network Services</strong></p>
<p>This module explains how to configure advanced features for Domain Name System (DNS) and Dynamic Host Configuration Protocol (DHCP). It also explains how to configure IP Address Management (IPAM) and Network Access Protection (NAP).</p>
<ul>
  <li>Implementing DNS and DHCP Enhancements</li>
  <li>Implementing IPAM</li>
  <li>Managing IP Address Spaces with IPAM</li>
  <li>NAP Overview</li>
  <li>Implementing NAP</li>
</ul>
<p><strong>Module 5: Implementing Remote Access</strong></p>
<p>This module covers the options for provisioning remote access with Windows Server 2012. It will specifically cover DirectAccess and VPNs and the considerations for implementing and managing these remote access technologies in your Windows Server 2012 environment, including the use of certificates.</p>
<ul>
  <li>Remote Access Overview</li>
  <li>Implementing DirectAccess by Using the Getting Started Wizard</li>
  <li>Implementing and Managing an Advanced DirectAccess Infrastructure</li>
  <li>Implementing VPN</li>
</ul>
<p><strong>Module 6: Implementing Failover Clustering</strong></p>
<p>This module explains how to provide high availability for network services and applications by implementing Failover Clustering. It will provide an overview of the terms and technologies as well as specific considerations and options for the services and applications under discussion.</p>
<ul>
  <li>Overview of Failover Clustering</li>
  <li>Implementing a Failover Cluster</li>
  <li>Configuring Highly-Available Applications and Services on a Failover Cluster</li>
  <li>Maintaining a Failover Cluster</li>
  <li>Implementing a Multisite Failover Cluster</li>
</ul>
<p><strong>Module 7: Implementing Hyper-V</strong></p>

<ul>

  <li>Configuring Hyper-V Storage</li>
  <li>Configuring Hyper-V Networking</li>
  <li>Configuring Hyper-V Virtual Machines</li>
