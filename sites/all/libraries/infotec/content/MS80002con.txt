<p><strong>Module 1: Configuring the Organizational Structure</strong>- This  module examines how to set up and manage business units.</p>

<p><strong>Module 4: Configuring System Settings</strong>- This module examines the series of configuration settings that are found within the  Microsoft Dynamics CRM application.</p>

<p><strong>Module 5: Change Management</strong>- This module examines best practices related to planning, executing, and testing changes in an implementation of Microsoft Dynamics CRM.</p>

<p><strong>Module 6: Report Customizations</strong>- This module examines options available for reporting in Microsoft Dynamics CRM and  basic report customization capabilities.</p>

<p><strong>Module 7: Performance and Maintenance</strong>-
