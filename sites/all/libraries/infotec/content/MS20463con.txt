<p><strong>Module 6: Debugging and Troubleshooting SSIS Packages</strong> - This module describes how you can debug packages to find the cause of errors that occur during execution. It then discusses the logging functionality built into SSIS that you can use to log events for troubleshooting purposes. Finally, the module describes common approaches for handling errors in control flow and data flow.</p>
<p>Lessons</p>
<ul>
  <li>Debugging an SSIS Package</li>
  <li>Logging SSIS Package Events</li>
  <li>Handling Errors in an SSIS Package</li>
</ul>
