This five-day instructor-led course provides students with the knowledge and skills to develop applications in the .NET Framework 3.5 using the C# 3.0 programming language. C# is one of the most popular programming languages in existence, and the C# 3.0 revision introduces new productivity, performance, functional programming and convenience features into the language.  This course features an overview of all language-related features, as well as an introduction to general .NET Framework features such as garbage collection, assembly loading, Reflection, Language-Integrated Query (LINQ) and many others.
