Extensible Markup Language (XML) is the universal format for data on the web. XML Foundations for Developers from LearnKey is a basic, general course for application developers, website builders, database administrators or anyone who needs to learn more about XML. Expert instructor Martin Schaeferle begins with XML basics and proceeds step-by-step to more advanced topics. At the conclusion of this course, you'll be able to make more informed decisions about how to solve business problems with XML.
