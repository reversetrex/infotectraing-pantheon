In this course you will learn to initiate enterprise projects and communicate through the project software with teams, collaborators, management and project stakeholders. Watch and learn as we teach the enterprise model in project management, how to share resources on multiple projects, build reporting using OLAP and SharePoint. Learn to administer the server for access, connectivity rights and version control. Work with the built in tools to create templates or implement custom projects. Learn to use the resource substitution wizard to staff projects and pool resources.
