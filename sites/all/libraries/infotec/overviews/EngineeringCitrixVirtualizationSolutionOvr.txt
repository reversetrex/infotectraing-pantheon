In this course, Citrix engineers will learn to plan for and perform a successful integration of the component technologies that make up a Citrix virtualization solution in a Windows Server 2008 environment. You will learn to interpret environmental assessment documentation, plan for a virtualization solution project, and integrate key Citrix virtualization technologies, including XenApp, XenServer, and XenDesktop, to create a holistic, end-to-end virtualization solution that will address today's most pressing business issues. The extensive, highly hands-on curriculum includes exercises that allow you to demonstrate the advanced technical knowledge you will gain in class.
