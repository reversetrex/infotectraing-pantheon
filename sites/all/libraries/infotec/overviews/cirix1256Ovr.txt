This course provides the necessary foundation to deploy and administer Citrix Presentation Server 4.0. This course covers installation and administration of Presentation Server and its components, including Web Interface, Secure Gateway, Installation manager and Load Manager. Students receive in-depth training using the Citrix Presentation Server Console to configure server farm and individual server settings and the Citrix Access Suite Console to manage multiple farms. In addition, this course provides training for managing, implementing and deploying the Citrix Presentation Server Clients.
