This course provides students with the knowledge and skills to design a complex data center and desktop architecture using Microsoft Windows Server 2008 with Hyper-V or Hyper-V Server 2008, Remote Desktop Services (RDS), System Center Suite, Microsoft Enterprise Desktop Virtualization (MED-V), Application Virtualization 4.6 (APP-V), as well as supporting applications and utilities.  The information included in this class will provide students with a solid baseline of technology solutions, architecture considerations, and management requirements that can be integrated to build a successful virtualization infrastructure.
