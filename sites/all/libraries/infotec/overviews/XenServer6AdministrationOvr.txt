This course provides the foundation necessary for administrators to effectively centralize and manage desktops in the datacenter and deliver them as a service to users anywhere. Students will learn how to select a desktop delivery model for each user group based on their needs, build proof-of-concept and production XenDesktop environments, assign virtual desktops to users and customize the user experience through policies. This course also provides instruction on how to build a Provisioning services farm to enable single-image desktop management, and how to host virtual machines on the client device using Citrix XenClient. This course prepares you for Exam A26 Citrix XenServer 6.0 Administration, a requirement for the CCA for Citrix XenServer 6 (CCA) certification.
