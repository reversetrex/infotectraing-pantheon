In this course, students will learn the basic concepts of relational databases and the SQL language. Participants will leave this course with the essential skills to write queries from both single and multiple tables, make changes to data in tables, and create and maintain database objects. Students also learn to control privileges for these objects.<br><br>
This course covers creating tables, indexes and constraints, and altering existing schema objects. Students also learn how to create and query external tables and use other advanced features of SQL in order to query and manipulate data within the database, use the dictionary views to retrieve metadata about their schema objects, and produce reports from applications. Students also learn about SQL functions and aggregate functions available in the Oracle database as well as how to use the regular expressions in SQL.
