JDBC 2.0 is a Java application programming interface that allows database developers to send SQL statements to virtually any relational database. In the Database Specialist Using JDBC 2.0 course from LearnKey, expert instructor Wayne Snyder uses examples, sample programs and clear step-by-step instruction to teach you database application design with Java. At the conclusion of this course, you will be able to write platform independent applications to access Oracle, Sybase, Informix, Microsoft SQL Server, and DB2 databases.
