LearnKey's Windows Server 2008 R2 Updates course is designed to give individuals insight into all of the new features of Windows Server 2008 R2 while taking into consideration the changes that have occurred in each of the Windows Server 2008 Microsoft MCTS and MCITP exams. This is not an all inclusive course, which means individuals planning to prepare for any of the Microsoft MCTS or MCITP Windows Server 2008 exams will also need to purchase and review the appropriate LearnKey course that maps to the specific Microsoft Windows Server 2008 exam that they are preparing for. This course covers exam objectives for exams; 70-640, 70-642 and 70-643. Exams 70-646 and 647 have not been updated by Microsoft.
