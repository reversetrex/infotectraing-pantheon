ACEAP is a 4-day, instructor-led, lecture / lab course that teaches you how to design, deploy, and optimize intelligent network services using the Cisco Application Control Engine (ACE) Appliance.    This course covers all of the key features of the ACE 2.0 software, including resource virtualization and management, server load balancing (Layer 2-4 and Layer 7), SSL termination and offload, and security features like application-layer inspection and fix ups.
