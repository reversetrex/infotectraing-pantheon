<p>This five-day instructor-led course will provide you with the knowledge and skills to configure and manage a Microsoft Exchange Server 2010 messaging environment. This course does not require previous Exchange Server experience, but does require that students have significant experience in managing Windows Server and Active Directory directory services or Active Directory Domain Services (AD DS). This course will teach you how to configure Exchange Server 2010, as well as provide guidelines, best practices, and considerations that will help you optimize your Exchange server deployment.</p>
