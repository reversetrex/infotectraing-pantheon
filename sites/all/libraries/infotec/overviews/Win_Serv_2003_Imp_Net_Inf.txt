The Windows Server 2003 Implementing Network Infrastructure course from LearnKey prepares you with the knowledge and skills needed to implement, manage, and maintain a Windows Server 2003 network. Expert instructor Aaron Spurlock covers critical topics like how to implement server IP addressing, name resolution, network security, routing and remote access. At the conclusion of the course, you will understand the skills required to implement a Windows Server 2003 network, and be prepared to pass MCP exam 70-291.
