The VB.NET 1.2 for Developers Part 1 course from LearnKey targets existing application developers who want to make the transition to developing for .NET. Expert instructors Andy Baron, Mary Chipman and Ken Getz show you how to use Windows forms and controls, apply object-oriented programming techniques, make use of XML web services and test and debug your applications. At the conclusion of this course, you'll be prepared to design, develop, and deploy a simple application using Visual Basic .NET.
