Learning Windows 7 training course from LearnKey will teach you basic techniques and concepts that will bridge your understanding with Microsoft's latest operating system: Windows 7. These topics include starting Windows 7, understanding what you can do with Windows 7, and new personalization techniques. Expert Ammon Wiese will introduce you to new and exciting features of Windows 7 such as Jump Lists, Aero Peek, Pinned Items, Homegroups and Microsoft's newest peer-to-peer network that allows users to share video, music, and other personal data quickly and securely.
