The Cisco IP Phone is a full-feature telephone providing voice communication over an Internet Protocol (IP) network.  This phone functions like a traditional analog phone allowing you to place and receive calls, setup speed dialing, redial, call transfer, conference calling and access to voice mail.  Because this phone is part of an IP, network it provides telephony services as well as data services.  This phone also provides a range of communication features that you could not expect with a traditional telephone.
