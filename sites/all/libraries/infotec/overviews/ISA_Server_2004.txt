ISA Server is a full-featured, application-layer firewall designed to protect organizations from attack by both external and internal IT security threats. In LearnKey's ISA Server 2004 course expert Aaron Spurlock covers everything from planning and installation to security configurations and remote monitoring to features of firewall integration, VPN architecture support, filtering methods and VPN traffic inspection capabilities. Detailed explanations provide a step-by-step walk through scenarios for any number of network configurations.
