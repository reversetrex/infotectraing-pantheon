This course provides the necessary foundation to effectively deploy and administer Citrix Presentation Server 4.5 and its components. This course covers installation and administration of Presentation Server and its components, including Load Manager, Installation Manager, Web Interface, Application Streaming and Secure Gateway. Students receive in-depth training using the Citrix Presentation Server and Presentation Server Clients and for using the various administrative consoles to configure published resources, policies, individual server and server farm settings, load evaluators, isolation environments, printers, streaming applications and much more.
