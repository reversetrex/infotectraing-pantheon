FrontPage 2003 is Microsoft's software for building professional websites. With the FrontPage 2003 course, you'll discover how the features and functionality of FrontPage 2003 can help you design and create dynamic and engaging web pages. Expert instructor Erin Olsen begins with website design and proceeds step-by-step to more advanced topics like database connectivity. At the conclusion of this course, you'll understand how the tools in FrontPage 2003 can help you build engaging, dynamic websites.
