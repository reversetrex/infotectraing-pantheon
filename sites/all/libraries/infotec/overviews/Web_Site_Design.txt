In this 7 session course Debbie will take you through the steps to build user friendly, interactive sites that visitors can easily navigate. You will learn important concepts about layout, color, and site planning. Explore with our expert how to use Photoshop, Flash, and Expression Web to make web sites come alive. Work with HTML, XHTML, CSS, and Div Tags. You will also be introduced to JavaScript and Java. Learn how to use Dreamweaver or Expression Web to build highly interactive web pages. This course will also prepare you to take the CIW Web Site Designer exam ID0-520.
