<p> After completing this workshop, students will be able to: </p>
<ul>
  <li> Connect to databases and read data. </li>
  <li> Query and update databases by using commands. </li>
  <li> Perform transactional operations. </li>
  <li> Perform disconnected operations programmatically. </li>
  <li> Perform disconnected operations by using Visual Studio 2005 wizards. </li>
  <li> Perform XML operations on disconnected data. </li>
  <li> Read and write XML data. </li>
  <li> Process XML data by using the Document Object Model DOM. </li>
</ul>
