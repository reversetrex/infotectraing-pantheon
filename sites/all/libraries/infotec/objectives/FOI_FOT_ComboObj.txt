<p><strong>FOI</strong></p>
<ul>
  <li>Properly assemble a patch cord using anaerobic epoxy that meets or exceeds TIA/EIA-568-B.3 loss requirements for a mated pair. (0.75 dB)</li>
  <li>Properly assemble a patch cord using oven-cured epoxy that meets or exceeds TIA/EIA-568-B.3 loss requirements for a mated pair. (0.75 dB)</li>
  <li>Properly assemble a mechanical splice that meets or exceeds TIA/EIA-568-B.3 attenuation requirement. (0.3 dB)</li>
  <li>Properly assemble a fusion splice.</li>
  <li>Measure bi-directional loss in a splice with a light source and power meter.</li>
  <li>Measure bi-directional loss in a fiber optic link with a light source and power meter using one of the TIA/EIA-526-14A methods.</li>
  <li>Measure the bi-directional loss in a mated pair with a light source and power meter.</li>
  <li>Properly evaluate a connector endface as defined by TIA/EIA-455-57B.</li>
</ul>
<p><strong>FOT</strong></p>
<ul>
  <li>Measure bi-directional loss in a fiber optic link with a light source and power meter using one of the TIA/EIA-526-14A methods.</li>
  <li>Perform measured patch cord testing with a light source and power meter.</li>
  <li>Perform loss per unit length acceptance testing in a fiber optic cable using an Optical Time Domain Reflectometer (OTDR).</li>
  <li>Locate a fault in a fiber optic cable using an Optical Time Domain Reflectometer (OTDR).</li>
  <li>Locate a mechanical interconnection and the end of the fiber in an Optical Time Domain Reflectometer (OTDR) trace.</li>
</ul>
