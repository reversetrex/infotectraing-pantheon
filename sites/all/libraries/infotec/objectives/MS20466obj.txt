<p>After completing this course, students will be able to:</p>
<ul>
  <li>Describe the components, architecture, and nature of a BI solution.</li>
  <li>Create a multidimensional database with Analysis Services.</li>
  <li>Implement dimensions in a cube.</li>
  <li>Implement measures and measure groups in a cube.</li>
  <li>Use MDX Syntax.</li>
  <li>Customize a cube.</li>
  <li>Implement a Tabular Data Model in SQL Server Analysis Services.</li>
  <li>Use DAX to enhance a tabular model.</li>
  <li>Create reports with Reporting Services.</li>
  <li>Enhance reports with charts and parameters.</li>
  <li>Manage report execution and delivery.</li>
  <li>Implement a dashboard in SharePoint Server with PerformancePoint Services.</li>
  <li>Use Data Mining for Predictive Analysis.</li>
</ul>
