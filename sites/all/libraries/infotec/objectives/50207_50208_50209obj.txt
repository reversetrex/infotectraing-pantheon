<p>After  completing this course, students will understand the basic technology behind  SharePoint and SharePoint sites, and know how to work in SharePoint Designer  2007 to create and modify SharePoint site content. Students will know how to  work with HTML pages, Web Part pages, and ASP.NET pages, and how to create  content based on existing master pages. Students will also know how to create  SharePoint lists and display them in Web Parts or as pages.</p>
