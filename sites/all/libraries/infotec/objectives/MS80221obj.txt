<ul>

    <li>Discuss the architecture and requirements of Microsoft Dynamics AX 2012.</li>

    <li>Understand the Microsoft Dynamics AX 2012 role based security architecture.</li>
    <li>Describe architecture for Services and the Application Integration Framework.</li>

    <li>Install services on IIS.</li>
    <li>Explain the capabilities of workflow in Microsoft Dynamics AX 2012.</li>
    <li>Configure an approval workflow.</li>
    <li>Use the SQL Server Analysis Services Project Wizard.</li>
    <li>Discuss the Microsoft Dynamics AX 2012 reporting architecture.</li>
    <li>Deploy Microsoft Dynamics AX 2012 SQL Server Reporting Services reports.</li>
    <li>Deploy and configure Enterprise Portal.</li>
