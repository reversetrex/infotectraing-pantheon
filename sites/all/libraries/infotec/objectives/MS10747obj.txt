<ul>

    <li>Discover and organize resources.</li>
    <li>Manage the Configuration Manager client.</li>
    <li>Manage inventory and software metering.</li>
    <li>Query and report data.</li>
    <li>Manage software deployments by using packages and programs.</li>
    <li>Create and deploy applications.</li>
    <li>Manage application deployment.</li>
    <li>Deploy and manage software updates.</li>
    <li>Implement System Center 2012 Endpoint Protection.</li>
    <li>Deploy operating systems by using System Center 2012 Configuration Manager.</li>
    <li>Manage compliance settings on devices.</li>
    <li>Manage mobile devices.</li>
