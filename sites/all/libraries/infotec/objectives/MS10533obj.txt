<p>After completing this course, students will be able to:</p>
<ul>
  <li>Describe how the features and architecture of Microsoft Lync Server 2010 enable real time communication and collaboration</li> 
  <li>Deploy Lync Server 2010</li> 
  <li>Configure users and rights using Role Based Access Control (RBAC)</li>
  <li>Deploy and manage Lync 2010 clients</li>
  <li>Configure and manage external user access using the Edge server role</li>
  <li>Deploy basic Enterprise Voice functionality</li> 
  <li>Extend Enterprise Voice functionality to include Trunk options, call park orbit, and the unassigned number feature</li>
  <li>Integrate Microsoft Exchange Server 2010 Service Pack 1 (SP1) Unified Messaging (UM) with Lync Server 2010</li>
  <li>Configure the Response Group Service, including agent groups, queues, and workflows</li>
  <li>Configure and manage conferencing</li>
  <li>Configure and use archiving and monitoring</li> 
  <li>Perform back up and restore of Lync Server 2010 critical data</li> 
  <li>Improve bandwidth management using Call Admission Control (CAC)</li> 
  <li>Configure and deploy Enhanced 9-1-1</li> 
  <li>Configure a Survivable Branch Server for a branch office</li>
</ul>
