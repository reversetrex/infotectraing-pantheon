<p> Unit objectives help students become comfortable with the course, and also provide a means to evaluate learning. Upon successful completion of this course, students will be able to: </p>
<ul>
  <li> Use protection and collaboration features </li>
  <li> Consolidate data from multiple worksheets </li>
  <li> Publish workbooks to the Web </li>
  <li> Import and export Excel data </li>
  <li> Use Excel's XML features to create XML maps and import XML data </li>
  <li> Modify Excel's default settings </li>
  <li> Customize toolbars and menus </li>
  <li> Create and editing macros </li>
</ul>
