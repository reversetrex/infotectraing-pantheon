<p>Students are starting to familiarize themselves  with print layout and design using InDesign. They would like to learn about the  tools and features available in InDesign CS3. In this course, they will work  with some of the tools and features to create eye-catching printed documents  using InDesign CS3. This course is intended for graphic designers, creative  professionals, print professionals, publishers, pre-press professionals, and  marketing communications professionals.</p>
